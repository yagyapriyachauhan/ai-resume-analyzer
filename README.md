# AI Resume Analyzer 🚀

An AI-powered resume analyzer built using MERN stack + Groq API.

## Features
- Resume Upload (PDF)
- Skill Detection
- ATS Score
- Role Matching
- AI-based Resume Analysis

## Tech Stack
- React
- Node.js
- Express
- Groq API

## Run Locally

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm run dev
